# CodeAlpha_handwritenCharacterRecognition
Handwritten Character Recognition using CNN and EMNIST Dataset

## Handwritten Character Recognition using CNN

This project recognizes handwritten characters using a Convolutional Neural Network (CNN) trained on the EMNIST Balanced dataset. It preprocesses images, trains the model, predicts characters, and evaluates performance using accuracy and loss graphs.

### Technologies Used
- Python
- TensorFlow/Keras
- Pandas
- NumPy
- Matplotlib

### Dataset
EMNIST Balanced Dataset  
https://www.kaggle.com/datasets/crawford/emnist

### Features
- Image preprocessing
- CNN-based character recognition
- Character prediction
- Accuracy and loss visualization
